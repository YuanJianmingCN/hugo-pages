树莓派部署 Hugo 的流程记录

July 24, 2025

背景#

最近我在树莓派 5B 上部署 Hugo 博客，并使用 CNB.cool 实现了内网博客的托管。

环境信息#

  • 树莓派 5B，Ubuntu 24.04 Server
  • Hugo v0.127.0（extended 版本）
  • 主题：PaperMod

安装步骤#

# 安装 Hugo
wget https://github.com/gohugoio/hugo/releases/download/v0.127.0/hugo_extended_0.127.0_Linux-64bit.tar.gz
tar -zxvf hugo_extended_0.127.0_Linux-64bit.tar.gz
sudo mv hugo /usr/local/bin/
安装最新版本
